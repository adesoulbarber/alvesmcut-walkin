# alvesmcut walk-in queue

v1 gratuite intelligente pour gérer une file d'attente salon barber.

## démarrage local
npm install
npm run dev

## déploiement vercel
- importer le dossier sur github
- connecter le repo à vercel
- build command: npm run build
- output: dist

## codes démo
- admin alves: alves2026
- admin adesoulbarber: adesoul2026
- barber alves: alves
- barber président: president
- barber adesoulbarber: adesoul

## important
cette v1 fonctionne en mode gratuit localstorage pour tester le design et le workflow. pour une vraie utilisation multi-appareils en salon, connecter supabase free plan avec le schema `supabase-schema.sql`.
